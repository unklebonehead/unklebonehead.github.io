I gave Ubuntu 23.04 a good try and I like it.

I gave Ubuntu 23.04 a good try and I like it. IMO it’s the best one so far. You dont have to use the snaps if you dont want. Just install Flatpak and boom. Get your work done. The new flutter installer worked like a charm on my machine except for not adding a swap partition. Two weeks solid as a daily driver for work and play and no issues after adding the swap.
